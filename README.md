# Wokwi-Chip-PWM
## Description

PWM Signal Generator ⎍⎍⎍⎍⎍

➤  0 - 10,100 Hz
➤  0 - 100 % duty 

The actual source code for the chip lives in [src/main.c](src/main.c), and the pins are described in [chip.json](chip.json).
